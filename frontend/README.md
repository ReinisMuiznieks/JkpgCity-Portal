## Installation

### Prerequisites

- Node.js (v18 or higher)
- npm (comes with Node.js)

### Install Dependencies

```bash
cd frontend
npm install
```

## Running the Server

### Development Mode

```bash
npm start
```

Server runs on: `http://localhost:8080`

```bash
npm run dev
```

Note: You need the backend running on http://localhost:3000 to fetch data
